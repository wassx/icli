# iCloud CLI

A command-line interface for interacting with iCloud services.

## Current Features
- Main menu with options for:
  - Mail
  - Calendar
  - iCloud Drive
  - Other services

## Installation

```bash
git clone https://github.com/yourusername/icli.git
cd icli
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
